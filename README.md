# blockstack-monero

Use your Blockstack account to easily send and receive Monero.

Uses the [MyMonero API](https://github.com/mymonero/hosted-monero-api-spec/blob/e4632679d46d3b6e2b5446c65f55bdacefdadac6/mymonero_api.md).
